VisionWatch – Driver Drowsiness Detection System

Developed a real-time driver drowsiness detection system using computer vision and Raspberry Pi to enhance road safety. The system monitors eye activity using Eye Aspect Ratio (EAR) and facial landmark detection.

When drowsiness is detected, alerts are sent wirelessly to a Raspberry Pi, triggering multi-stage voice warnings. The system resets automatically once the driver regains alertness.
